InvocandoJS

Un grimorio moderno para hacer magia con JavaScript

📚 Documentación 🐙 GitHub

Diseñado para validar formularios, manipular la interfaz y reunir funciones que realmente usas. Pensado para enseñar, para simplificar, y para hablar en tu idioma sin perder poder. Es modular, claro, y abierto: trae un validador desacoplado y funciones sueltas listas para invocar. ¿Tienes tu propio hechizo? Guárdalo aquí. Llévatelo contigo. Así funciona este grimorio.
